Lyricit

Add lyrics to your itunes songs through command line.

Installation

$ gem install lyricit

Usage

To update the lyrics for current song.

$ lyricit -c

To update the lyrics for a particular song

$ lyricit -s somebody

Above command will give you the list of songs matching the somebody
